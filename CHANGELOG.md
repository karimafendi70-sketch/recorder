# Changelog

## v1.0.0

- Multi-day surf overview met dagcards, dagdelen en consistente slotselectie.
- Slotdetailpaneel met condition-tags, surf-advies en filter-aware lege toestanden.
- Multi-spot overzicht met beste spots voor de actieve dag en score per spot.
- Dagelijks surf report per spot/dag met compacte tekstsamenvatting.
- PWA-basis (installable shell + service worker) en lichte performance-optimalisaties.
- Tooling en QA-basis met helper-tests (`npm run test-helpers`) en lint (`npm run lint`).
