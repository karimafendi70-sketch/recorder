# FreeSurfCast

A profile-driven surf-forecast web app that scores 31 European spots using live Open-Meteo data and your personal surf preferences.

**Stack:** Next.js (App Router) · TypeScript · React · Leaflet · Pure CSS / CSS Modules · Open-Meteo API (no key required)

## Quick Start

```bash
npm install
npm run dev        # http://localhost:3000
```

## Features

- 31 surf spots with live marine + weather forecasts
- 0–100 scoring engine tuned to your skill, wave range and condition weights
- SpotSearchBar, SpotBrowser (country / difficulty / type filters), favorites & recent
- Interactive Leaflet map with deep-link into the forecast page
- Multi-spot Insights: best-spot badge, ranking, heatmap, timeline
- Full EN/NL i18n, responsive mobile-first layout
- Graceful loading & error handling with automatic mock-data fallback
